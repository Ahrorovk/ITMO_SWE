# Лабораторная работа №4 — Аппроксимация функции МНК
## Вариант 1: y = 2x/(x⁴+1), x ∈ [0, 2], h = 0.2

### Структура проекта

```
lab4/
├── src/
│   ├── main/kotlin/approximation/
│   │   ├── Approximation.kt   # Все методы МНК (линейный, поли-2, поли-3,
│   │   │                      #   экспоненциальный, логарифмический, степенной)
│   │   ├── Output.kt          # Форматированный вывод и ASCII-графики
│   │   └── Main.kt            # Точка входа
│   └── test/kotlin/approximation/
│       └── ApproximationTest.kt  # 12 юнит-тестов (JUnit 5)
├── build.gradle.kts
├── settings.gradle.kts
├── gradlew  /  gradlew.bat
└── report/
    ├── report.tex             # Отчёт LaTeX (XeLaTeX)
    └── report.pdf             # Скомпилированный PDF
```

### Требования

- JDK 11 или новее (JDK 17/21 рекомендуется)
- Интернет-соединение при первом запуске (Gradle скачает Kotlin compiler)

### Сборка и запуск

```bash
# Запустить программу
./gradlew run

# Запустить с сохранением отчёта в файл
./gradlew run --args="result.txt"

# Запустить тесты
./gradlew test

# Собрать fat-jar
./gradlew jar
java -cp build/libs/lab4-approximation-1.0.0.jar approximation.MainKt
```

### Что выводит программа

1. Таблицу исходных данных (11 точек)
2. Коэффициент корреляции Пирсона для линейной зависимости
3. Сводную таблицу всех 6 аппроксимаций (S, δ, R²), отсортированных по δ
4. Детальные таблицы отклонений для линейной и квадратичной функций
5. ASCII-график всех функций
6. Полный отчёт в файл (по умолчанию `output_report.txt`)
